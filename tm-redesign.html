TM Education - Re-Design
Project Overview

This project aimed to re-work the existing TransferMate Education [Links / Web Solution] with an up to date design and user experience that creates delight and ease of use for the millions of users across the globe. Previously, the design language, user experience and overall structure of the links which TM provvided for each client was really bad and was full of clutter, which created confusion and was rough to navigate in general.

With this project we followed trends and best use practices to ensure that the TM Education [Links / Web Solution] are up to speed and on par with competitors in terms of user experience and design.
